About Me

Hi, I'm Alex Turner, a passionate fullstack developer with over 5 years of experience building dynamic web applications. I specialize in JavaScript ecosystems, including React for frontend and Node.js with Express for backend development. My expertise also extends to databases like MongoDB and PostgreSQL, and I'm proficient in modern DevOps practices using Docker and AWS. I'm committed to writing clean, scalable code and continuously learning new technologies to deliver high-quality solutions. When I'm not coding, you can find me contributing to open-source projects or exploring new frameworks and libraries.

Projects I've Undertaken

  • E-Commerce Platform: Developed a full-featured online store with React, Redux, Node.js, and MongoDB, including user authentication, payment processing, and admin dashboard.
  • Task Management App: Created a collaborative task management tool using Vue.js, Express, and PostgreSQL, featuring real-time updates with WebSockets.
  • Health Tracking Dashboard: Built a responsive dashboard for fitness data using React, D3.js for visualizations, and a RESTful API with Python Flask.
  • Blog CMS: Designed and implemented a content management system with Next.js, GraphQL, and MongoDB, supporting markdown editing and SEO optimization.
  • Chat Application: Engineered a real-time messaging app using Socket.IO, React Native for mobile, and a scalable Node.js backend.
