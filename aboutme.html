About Me

Hi, I'm Alex Turner, a passionate fullstack developer with over 5 years of experience building dynamic web applications. I specialize in JavaScript ecosystems, including React for frontend and Node.js with Express for backend development. My expertise also extends to databases like MongoDB and PostgreSQL, and I'm proficient in modern DevOps practices using Docker and AWS. I'm committed to writing clean, scalable code and continuously learning new technologies to deliver high-quality solutions. When I'm not coding, you can find me contributing to open-source projects or exploring new frameworks and libraries.
